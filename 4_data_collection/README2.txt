To use this script:

1) Download XAMPP or any other similar software and run the database as described in the documentation to run the database. (PHPMyAdmin Installation Guide: https://fluentsupport.com/phpmyadmin-installation/)
2) Make sure MySQL and Apache are running!
3) Within the script, make sure your database name, username and password match.
4) Run the script. It will generate a spreadsheet with restock information.

Information:

Product's name
Number of restock
Percentage of restocks
A pie chart for visual representation
